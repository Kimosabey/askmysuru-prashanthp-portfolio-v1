# V2 Portfolio Architecture & Routing Plan

## Objective
To seamlessly adopt the structural navigation menu from the legacy `profile.html` reference and inject it with the verified, professional content from the `PRASHANTH_P_DOSSIER.md`.

## Navigation Mesh Upgrades Required
The `TopNavBar.tsx` must be updated to include the following core routing structure:
1. `Home` (Reel / Foundation)
2. `About` (The Media House Builder)
3. `Cinematography` (The Archives)
4. `Gallery` (Visual Stills & BTS)
5. `Certificates` (Awards & Distinctions)
6. `Clients` (Brand Ecosystem & Partners)
7. `Contact` (Professional Inquiries)

## Execution Strategy
To ensure maximum scalability and clean code, we are documenting the exact copy, components, and data structures required for each of these 6 new/upgraded routes inside this `docs/` folder. This ensures the UI/UX team can build the specific Next.js `.tsx` components flawlessly using established variables.

**Refer to the sequential `.md` files in this directory for exact page-level implementation specifications.**
