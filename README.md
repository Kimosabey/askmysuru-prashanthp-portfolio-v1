# 🎬 Prashanth P. — The Cinematographic Portfolio V1

**Media House Builder · Visionary Director · Heritage Documentarian**

A premium, high-fidelity digital dossier built for **Prashanth P.**, Founder of [Ask Mysuru](https://askmysuru.com) and [Political360](https://political360.in). This portfolio is engineered to document his journey as an award-winning director and media house builder, blending academic rigor with cinematic storytelling.

## ✨ Premium Features

- **🛡️ Cinematographic Viewfinder Cursor**: A sophisticated, custom-engineered cursor that adapts to interactive elements with a camera-viewfinder aesthetic.
- **🌓 Studio Dynamic Themes**: High-contrast "Cinematic Dark" and "Studio Light" modes tailored for maximum readability and visual impact.
- **🎞️ Session-Aware Preloader**: An optimized entry experience that simulates 35mm film processing, configured to run only once per session for seamless site-wide navigation.
- **📐 Fluid Typography System**: Perfectly scaled typography (Fluid H1–H3) ensuring pixel-perfect consistency from mobile to desktop.
- **🎨 Branding Continuity**: Theme-aware logo transitions and branding assets that adapt to the user's selected environment.

## 🛠️ Technology Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (Newest Engine)
- **Animation**: [Framer Motion](https://framer.com/motion/)
- **Theming**: [next-themes](https://github.com/pacocoursey/next-themes)
- **Typography**: Playfair Display (Headline), Outfit (Body/Label), Noto Sans Kannada.
- **Smooth Scroll**: [Lenis](https://lenis.darkroom.engineering/)

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Build for production
npm run build
```
